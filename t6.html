Боксит кальцинированный

Кальцинированный боксит производят посредством обжига/спекания/кальцинации маложелезистого и малощелочного сырого боксита при температуре 1600 - 1800 °C. При этом образуются высокоогнеупорные минеральные фазы корунд и муллит. Кальцинированный боксит - это один из важнейших сырьевых материалов для производства формованных и неформованных огнеупоров для сталелитейной промышленности, для литейных производств, цементных и стекольных заводов. Боксит может поставляться "прямо из печи" как фракционированное сырьё или по требованию клиента в виде разных фракций и разного размера помола. 0 – 1 мм.

Химический анализ: Al2O3 75- 92 %
Плотность: 2,90 - 3,40 r/cm³
Бокситовый заполнитель фракция: (0-0,3 мм, 0-0,5 мм, 0-1 мм, 1-3 мм, 3-5 мм, 5-8 мм.)
Бокситовый порошок: Mesh100; Mesh200; Mesh 325.

Применяется:

    — Производство бокситового цемента
    — Абразивы
    — Керамическая промышленность
    — Химическая промышленность для всех видов соединений алюминия
    — Металлургия алюминия
    — Прецизионное литье / литье по выплавляемым моделям: мелкие фракции бокситового грога можно превратить в формы для точного литья
    — Огнеупор: алюмосиликатное огнеупорное волокно, смесь магнезии и абукситового грога со связующими для заливки в ковш расплавленной стали
Упаковка:
    — Мешок по 25 кг на деревянном поддоне
    — Мешок биг бег по 1000 кг
